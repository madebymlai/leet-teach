# Mission: LeetCode DSA Practice with AI Coaching

## Why
Become proficient in data structures and algorithms for technical interviews and general problem-solving ability. Use a terminal-based workflow (helix + leetcode-cli + MCP) with an AI coach providing progressive hints and pattern recognition.

## Success looks like
- Can independently solve medium-level problems in under 30 minutes
- Recognize common patterns (two pointers, sliding window, BFS/DFS, DP, etc.)
- Can analyze time and space complexity for any solution
- Have a structured approach to new problems (understand → plan → code → test → optimize)

## Constraints
- Terminal-first workflow (helix editor, no browser IDE)
- AI coach provides hints, not full solutions
- Focus on understanding patterns, not memorizing solutions

## Out of scope
- Competitive programming speed optimization
- System design
- Language-specific trivia
