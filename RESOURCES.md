# LeetCode DSA Resources

## Knowledge

- [Book: _Grokking the Coding Interview_ — Design Gurus](https://designgurus.io/course/grokking-the-coding-patterns)
  Pattern-based approach to coding interviews. Use for: learning the 15-20 common patterns.
- [NeetCode 150](https://neetcode.io/)
  Curated problem list organized by pattern with video solutions. Use for: structured practice progression.
- [visualgo.net](https://visualgo.net/)
  Visualizations of data structures and algorithms. Use for: understanding how algorithms work step-by-step.

## Wisdom (Communities)

- [r/leetcode](https://reddit.com/r/leetcode/)
  Active community for problem discussion and interview prep. Use for: alternative approaches, contest discussion.
- [LeetCode Discuss](https://leetcode.com/discuss/)
  Official problem discussion forums. Use for: reading optimized solutions after solving.

## Gaps

- No structured spaced repetition system integrated yet (consider leetsolv CLI)
- No progress tracking dashboard (leetcode stat provides basic stats)
